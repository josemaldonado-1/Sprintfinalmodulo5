
SPRINT FINAL MODULO 5 


ASESORIAS EN PREVENVION DE RIESGOS 

El proyecto presentado conciste en una aplicacion web que permite al administrador del sitio
luego de validar sus credenciales, crear nuevos usuarios en las categorias Cliente,Administrativo
y profecional, ademas de poder crear capacitaciones.
Estos datos son almacenados en una base de datos la cual puede ser editada. una vez terminada la operacion
los datos son almacenados en sus respectivas tablas al mismo tiempo que se muestran por pantalla.
la pagina es responsiva, incluye un menu y un formulario de contacto el cual tambien es almacenado
